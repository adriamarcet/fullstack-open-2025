# KELP (KelpUI)

Official docs: https://kelpui.com/docs/getting-started/installation/

Quick installation options used in this project:

- CDN (added to `index.html`):

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/kelpui@latest/dist/kelp.min.css" />
```

- GitHub install (alternative):

```bash
npm install github:cferdinandi/kelp --save
```

Usage notes:

- KELP is a minimal CSS utility and component library. You can apply KELP classes directly in JSX markup.
- See the official docs for components, utilities, and best practices: https://kelpui.com/docs/getting-started/installation/
